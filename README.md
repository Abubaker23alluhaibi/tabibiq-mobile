# TabibiQ Mobile App

تطبيق TabibiQ المحمول - منصة طبيب العراق

## المميزات

### للمستخدمين
- **حجز المواعيد مع الأطباء**: حجز مواعيد بسهولة مع أفضل الأطباء
- **البحث والفلترة**: بحث متقدم عن الأطباء حسب التخصص والموقع
- **عرض محدود للأطباء**: عرض 10 أطباء فقط في الشاشة الرئيسية (المميزين أولاً)
- **صفحة كل الأطباء**: عرض جميع الأطباء مع إمكانية البحث والفلترة المتقدمة
- **إدارة الملف الشخصي**: تحديث المعلومات الشخصية
- **نظام الإشعارات**: إشعارات فورية للمواعيد والتذكيرات
- **تذكيرات الأدوية**: نظام تذكير ذكي للأدوية

### للأطباء
- **لوحة تحكم شاملة**: إدارة المواعيد والإحصائيات
- **إدارة المواعيد**: عرض وتعديل وإلغاء المواعيد
- **تقويم المواعيد**: عرض جميع المواعيد في تقويم شهري
- **إدارة أوقات العمل**: تحديد أيام وساعات العمل
- **إحصائيات متقدمة**: تحليل الأداء ومعدلات الحضور
- **إدارة الحضور**: تسجيل حضور المرضى

### الميزات التقنية
- **دعم متعدد اللغات**: العربية، الكردية، الإنجليزية
- **واجهة مستخدم حديثة**: تصميم عصري وسهل الاستخدام
- **أداء محسن**: استجابة سريعة وتحديث فوري
- **أمان عالي**: حماية البيانات والمصادقة الآمنة

## التحديثات الجديدة

### 1. تحسين عرض الأطباء في الشاشة الرئيسية
- **عرض محدود**: عرض 10 أطباء فقط (المميزين أولاً)
- **ترتيب ذكي**: الأطباء المميزين يظهرون أولاً
- **عرض تدريجي**: إمكانية عرض المزيد من الأطباء
- **أداء محسن**: تحميل أسرع للشاشة الرئيسية

### 2. صفحة كل الأطباء المحسنة
- **بحث متقدم**: البحث بالاسم والتخصص والموقع
- **فلترة ذكية**: فلترة حسب التخصص والمحافظة
- **عرض محسن**: معلومات إضافية مثل التقييم والخبرة
- **تجربة مستخدم محسنة**: واجهة سهلة الاستخدام

## التثبيت والتشغيل

### المتطلبات

- Node.js 18+
- npm أو yarn
- Expo CLI

### التثبيت

```bash
# تثبيت التبعيات
npm install

# تشغيل التطبيق
npm start
```

### البناء

```bash
# بناء لنظام Android
npm run build:android

# بناء لنظام iOS
npm run build:ios

# بناء للمعاينة
npm run build:preview
```

## النشر

يتم النشر باستخدام Expo EAS Build:

```bash
# نشر الإنتاج
eas build --platform all --profile production

# نشر المعاينة
eas build --platform all --profile preview
```
